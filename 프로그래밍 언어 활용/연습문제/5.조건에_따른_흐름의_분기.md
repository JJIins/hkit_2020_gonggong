### [문제1]
```
1부터 99사이에 있는 정수 중에서 7의 배수이거나 9의 배수인 정수를 출력하는 프로그램을 작성
해 보자. 단! 7의 배수이자 9의 배수인 수는 한번만 출력해야 한다.
```

### [문제2]
```
두 개의 정수를 입력받아서 두 수의 차를 계산하는 프로그램을 구현해 보자. 단 무조건 큰 수에서 
작은 수를 뺀 결과를 출력해 줘야 한다. 예를 들어서 입력된 두 개의 수가 순서대로 12, 5라면 12-
5를 계산해야 하고, 4, 15이라면 16-4를 계산해야 한다. 따라서 출력 결과는 무조건 양수가 될 것
이다.
```

### [문제3]
```
구구단을 출력하되 짝수단(2단, 4단, 6단, 8단)만 출력을 하게끔 하자. 또한 2단은 2x2까지, 4단
은 4x4까지, 6단은 6x6까지, 8단은 8x8까지만 출력하게끔 프로그램을 구현해 보자. 이렇게 어
수선한 문제를 제시한 이유는 continue와 break문의 사용법을 익히기 위해서다. 그러니 가급적
이면 continue와 break문을 사용해서 문제를 해결하기 바란다.
```

### [문제4]
```
다음 식을 만족하는 모든 a와 b를 구하는 프로그램을 구현해 보자.

  ab
 +ba
-----
  99

가능한 모든 수의 조합을 시도해 보기 위해서, for문과 같은 반복문을 2중으로 중첩시켜야 한다.
여려분을 조금 더 생각하게 하는 문제이므로 해결책이 바로 떠오르지 않는다고 괴로워하지 말자.
시간을 두고 천천히 고민해 볼 문제이다. 출력 결과의 예로 18+81, 27+72 등등이 있다.
```
